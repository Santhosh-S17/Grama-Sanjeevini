# Grama-Sanjeevini 🏥

Grama-Sanjeevini is a Rural Pharmacy Network Android application built using Kotlin, Jetpack Compose, Firebase, and GenAI concepts.

The app helps villagers search medicines across nearby pharmacies and allows pharmacy owners to manage medicine inventory in real time.

---

# ✨ Features

## 👨‍🌾 User Features
- Search medicines instantly
- View nearby pharmacies
- Emergency medicine detection
- Real-time medicine updates
- Distance-based pharmacy listing

## 🏪 Pharmacy Owner Features
- Owner login
- Add medicines
- Update stock
- Delete medicines
- Expiry alerts
- Emergency medicine tagging

# 🎯 Problem Solved

Villagers often travel long distances searching for medicines.

Grama-Sanjeevini connects nearby pharmacies into a shared rural healthcare network helping users:
- Find medicines quickly
- Reduce travel
- Reduce medicine wastage
- Improve emergency access
